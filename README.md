# 📦 Boxpad Pro - Enterprise Dashboard

A premium, high-end React dashboard featuring **Bento Grid** layouts, **Glassmorphism** aesthetics, and high-performance animations.

## 🚀 Live Demo
https://my-hiring-proj.netlify.app/

## 📸 Preview

## ✨ Features
- **Frontend Architect Identity:** Designed with a focus on visual storytelling.
- **Glassmorphic UI:** Premium look with blurred backgrounds and neon glows.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
- **Dynamic Loading:** "Extracting Information" sequence with animated honeycombs.

## 🛠️ Tech Stack
- **Library:** React.js + TypeScript
- **Styling:** CSS3 (Custom Properties & Animations)
- **Icons:** Lucide React
- **Animations:** Framer Motion (optional)

## 🏗️ Setup Instructions
1. Clone the repo: `git clone https://github.com/Tajamull/Dana-s-Projects.git`
2. Install dependencies: `npm install`
3. Run project: `npm start`
4. Install lucide-reacts
5. install react-icons
